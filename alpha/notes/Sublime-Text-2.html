Notes on Sublime Text 2

Sep 23rd, 2012

An editor perhaps better than any

Download Sublime Text 2 for linux/mac/windows, or [download the dev build](Sublime Text 2.

Sublime Text 2 may be downloaded and evaluated for free, however a license must be purchased for continued use. There is currently no enforced time limit for the evaluation.

However, I bought a liscence as other ST2 lovers did.

A Review on Vimeo

Tutsplus has a nice series of tutorial on Sublime Text 2: Perfect Workflow in Sublime Text 2, check it out!

1. Change The Icon

Perhaps you’d like to change the default icon of sublime (especially true if you’re a mac user):

  • dmatarazzo’s pack https://github.com/dmatarazzo/Sublime-Text-2-Icon

  • Nate Beaty’s alternative: http://www.sublimetext.com/forum/download/file.php?id=200

Just cd /Applications/Sublime\ Text\ 2.app/Contents/Resources/, and replace the file “Sublime Text 2.icns”

2. Soda Theme & Solarized Color Scheme

Soda Theme: https://github.com/buymeasoda/soda-theme Solarized Color Scheme is in the “Color Schemes - Default” directory, and git repo is: https://github.com/deplorableword/textmate-solarized

Modify the file ~/Library/Application\ Support/Sublime\ Text\ 2/Packages/Color\ Scheme\ -\ Default/Solarized\ (dark).tmTheme:

3. OSX Command Line Tools

subl command arguments are:

  • –project : Load the given project
  • –command : Run the given command
  • -n or –new-window: Open a new window
  • -a or –add: Add folders to the current window
  • -w or –wait: Wait for the files to be closed before returning
  • -b or –background: Don’t activate the application
  • -s or –stay: Keep the application activated after closing the file
  • -h or –help: Show help (this message) and exit
  • -v or –version: Show version and exit

4. Open with Sublime Text 2 in Finder (or Path Finder)

Git Repo: https://github.com/fallroot/open-with-sublime-text-2

6. CodeIntel

Git Repo: https://github.com/Kronuz/SublimeCodeIntel

Sublime CodeIntel will allow you to jump around symbol definitions even across files with just a click. To “Jump to Symbol Declaration” use super+f3 or alt+click over the symbol. Start typing code as usual, autocomplete will pop up whenever it’s available. To trigger manual codeintel autocompletion use shift+ctrl+space.

7. CTags

CTags let you quickly navigate to code definitons (classes, methods, etc.). First:

brew install ctags or sudo port install ctags

Check out the default CTags key bindings to see what you can do.

8. Indent Guides

Git Repo: https://github.com/SublimeText/IndentGuides

9. Multi-Selection:

  • Press Alt or Command and then click in each region where you require a cursor.
  • Select a block of lines, and then press Shift + Command + L.
  • Place the cursor over a particular word, and press Control/Command + D repeatedly to select additional occurrences of that word.
  • Alternatively, add an additional cursor at all occurrences of a word by typing Alt+F3 on Windows, or Ctrl+Command+G on the Mac. Amazing!!
